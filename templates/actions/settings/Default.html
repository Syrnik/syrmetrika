Название счетчика

Название переменной счетчика Метрики. Обычно выглядит как yaCounterXXXXX, где XXXXX цифры. Оставьте это поле пустым, чтобы отключить плагин на выбранной витрине
Название цели «Корзина»

Настроенное название цели для посещения страницы «Корзина». Оставьте это поле пустым, если не надо учитывать достижение этой цели
Название цели «Контактная Информация»

Настроенное название цели для посещения страницы оформления заказа с вводом контактной информации. Оставьте это поле пустым, если не надо учитывать достижение этой цели
Название цели «Выбор доставки»

Настроенное название цели для посещения страницы оформления заказа с выбором способа доставки. Оставьте это поле пустым, если не надо учитывать достижение этой цели
Название цели «Выбор оплаты»

Настроенное название цели для посещения страницы оформления заказа с выбором способа оплаты. Оставьте это поле пустым, если не надо учитывать достижение этой цели
Название цели «Подтверждение заказа»

Настроенное название цели для посещения страницы оформления заказа с подтверждением заказа. Оставьте это поле пустым, если не надо учитывать достижение этой цели
Название цели «Заказ оформлен»

Настроенное название цели для посещения страницы успешным окончанием оформления заказа. На этом этапе в Метрику будут дополнительно передан список заказанных товаров и номер заказа. Оставьте это поле пустым, если не надо учитывать достижение этой цели
